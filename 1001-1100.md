# 1015. Smallest Integer Divisible by K
看某个余数是否已经出现，出现过说明要开始循环节了

除去`2`和`5`，其他都有解

另有等价的两个式子，后面的式子可以解决溢出问题
``` java
while (x % k != 0) {
    x = x * 10 + 1;
}
```
``` java
while (x % k != 0) {
    x = x % k;
    x = x * 10 + 1;
}
```
