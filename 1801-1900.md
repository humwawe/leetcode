# 1863. Sum of All Subset XOR Totals
枚举子集

`O(n)`的做法是对每一位，考虑有多少个可以为`1`，则结果为`count*(1<<i)`，`count`的值可由组合数求出为`2^(n-1)`
``` java
for (int num : nums) {
    res |= num;
}
return res << (nums.length - 1);
```

