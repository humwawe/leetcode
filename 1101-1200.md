
`dp[i][j]`，表示到`i`和到`j`的时候（可不包含）的最长公共前缀
``` java
for(int i = 1; i<=m; i++){
    for(int j = 1; j<=n; j++){
        if(text1.charAt(i-1) == text2.charAt(j-1)){
            dp[i][j] = dp[i-1][j-1] + 1;
        }else{
            dp[i][j] = Math.max(dp[i-1][j],dp[i][j-1]);
        }
    }
}
```
